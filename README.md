# AI 聊天页面

一个精美的 AI 聊天界面，支持多种 AI 模型切换。

## 功能特点

- 现代化的用户界面，使用 React 和 Tailwind CSS 构建
- 支持多种 AI 模型切换
- 实时聊天界面
- 响应式设计，适配各种设备
- 打字动画效果
- 消息复制、点赞功能

## 技术栈

- React.js - 前端框架
- Webpack - 构建工具
- Tailwind CSS - 样式框架
- PostCSS - CSS 预处理器

## 安装和运行

1. 克隆仓库
   ```
   git clone https://github.com/lqwlove/ai-start-page.git
   cd ai-start-page
   ```

2. 安装依赖
   ```
   npm install
   ```

3. 启动开发服务器
   ```
   npm start
   ```

4. 构建生产版本
   ```
   npm run build
   ```
